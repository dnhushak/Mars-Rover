unsigned distancecalc ( int  time)

Distance Calculator.

Calculates distance from time (34,000 cm/seconds = 34 cm/ microsecond, divided by 16 because the prescalar of 1024 causes the timer to increment every 1/15625 of a second We're trying to get microseconds, so divide that 15625 by 1000, so the timer increments every 1/15.6 of a microsecond, which is close to 16, so divide by 16. For some unexplained reason, I had to divide by 2 to get calibration correct.

Parameters
timeTime in an integer value from the ping sensor (from rising edge to falling edge) - represents timer ticks
Returns
distance in centimeters
